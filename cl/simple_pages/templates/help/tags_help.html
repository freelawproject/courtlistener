Help with Tags on CourtListener

The tagging feature on CourtListener helps you to tag and organize cases that you are working with.

Currently you are only able to tag dockets, but we plan to add documents and other types of records in the future.

We hope that if you find our tags useful, you will donate to Free Law Project, the non-profit supporting CourtListener and RECAP, to support the ongoing development and maintenance of this and other features.

Tags Overview

Every tag you create has a page on CourtListener listing items that are labelled with that tag. This allows you to organize and describe our content for your own purposes or to share with others. When you are on one of your tag pages, you can add a description of the tag, and you can make the tag public so that other people can see it too.

If you do not make a tag page public, nobody else will know you have that tag. Once a tag is made public it is immediately visible to others until you make it private again. All of your public tags will be visible to others as a {% if user.is_authenticated %}list of tags in your profile{% else %}list of tags in your profile{% endif %}.

Tagging a Docket

Tags can be added to any docket by clicking the Tags button on a docket. Once clicked, a tagging interface will appear:

In that window, you can create new tags by typing in a name and pressing enter. If that tag already exists, it will be checked or unchecked when you press enter. If the tag does not yet exist, it will be created and the docket will be tagged with it. When you're done creating tags, close the tagging tool.

Editing or Deleting a Tag

Tags can be edited or deleted. To edit a tag, either find it on {% if user.is_authenticated %}your list of tags{% else %}your list of tags{% endif %} or open it via the icon in the tagging tool.

When editing a tag, you can add a description to it using our editor or you can make it either public or private by toggling the setting in the actions drop down. Public tag pages are immediately visible to others.

Tags can be deleted either from the list of tags in your profile page or by using the actions drop down on a tag page, as shown above.
